Preentrega proyecto coder:

La web es sobre una cerveceria artesanal;
Se asigna como clases al personal que pertenece a la misma, a los productos que venden y a los estilos de uno de los productos (cerveza).



urls para navegar por el sitio:


 
path "http://127.0.0.1:8000/cerveceria/inicio"   --> Pagina de inicio
path "http://127.0.0.1:8000/cerveceria/estilos_formulario"  --> Formulario para agregar nuevos estilos a la BD
path "http://127.0.0.1:8000/cerveceria/productos_formulario"  --> Formulario para agregar nuevos productos a la BD
path "http://127.0.0.1:8000/cerveceria/staff_formulario"  --> Formulario para agregar nuevo personal a la BD
path "http://127.0.0.1:8000/cerveceria/lista_estilos"  --> Enlista los estilos con sus atributos
path "http://127.0.0.1:8000/cerveceria/lista_staff"  --> Enlista al personal con sus atributos
path "http://127.0.0.1:8000/cerveceria/lista_productos"  --> Enlista los productos con sus atributos
path "http://127.0.0.1:8000/cerveceria/busquedaAlcohol"  --> Formulario para buscar todos los estilos para un cierto grado de alcohol
path "http://127.0.0.1:8000/cerveceria/buscar/"  --> Resultado de la busqueda



se deja un ejemplo para asignar un estilo a la BD

nombre = "PORTER" , ibu = 40 , alcohol = 6 , cuerpo = "Pleno", amargor = "Medio", aroma = "Caramelo", temp_cons = 8
